strong 태그

<strong> creating web pages</strong>
strong은 HTML에서 문법적으로 태그
앞에 있는 태그를 열린 태그, 뒤에 있는 태그를 닫히는 태그(/)
strong이라는 것은 creating web pages가 중요하다고 설명

h태그

h뒤에는 숫자가 붙는다. 숫자가 커지면 글씨가 작아진다. h를 쓰면 글씨가 두꺼워진다.
The <h1> to <h6> tags are used to define HTML headings.
<h1> defines the most important heading. <h6> defines the least important heading.
heading은 제목이라는 뜻, h1은 가장 큰 제목 태그

줄바꿈을 한 이유는 단락을 표현하기 위해서 그런데 단락 태그가 있다.
html paragraph tags로 검색(html 단락 태그)
열리는 태그와 닫히는 태그가 존재
p는 단락을 구분해줌
br태그는 쓰는만큼 줄바꿈이 되기 때문에 원하는 만큼 간격을 줄 수 있는 장점이 있음.

title태그

파일명이 제목임 이것을 <title>WEB1 - html</title>
title을 꼭 써야함. 검색엔진도 title을 책표지같은 정보로 사용함.

META태그

한글로 작성했는데 웹페이지에서는 제대로 출력되지 않음.
이 파일은 UTF-8형식으로 저장됨.
따라서 웹브라우저가 열때도 UTF-8로 열어야 문제가 없음.
<metacharset="UTF-8"> -> utf-8형식으로 문서를 읽어라

body태그, head태그, html태그

title과 meta charest은 밑에 본문을 설명해주는 것들 따라서 구분해주어야 함.
본문은 body을 설명해주는 것은 head로 감싼다.
body와 head는 고위층 태그인데 이것을 감싸는 고위층 태그 html태그가 있음.
그리고 html태그 위에 관용적으로 이 문서는 html로 만들어졌다. <!doctype html>

a태그

a태그를 통해 웹페이들을 발견할 수 있고, 검색엔진이 존재
HTML의 약자 HyperText Markup Language의 HyperText가 바로 a 태그를 의미
anchor의 첫글자 a 앵커는 배가 정박할 때 사용하는 닻을 의미
a태그는 링크를 표현하는 기능이다.
링크는 태그명이 a이다. <a href="http://www.icqa.or.kr/cn/page/network" target="_blank" title="네트워크관리사홈페이지"> h는 Hyper Text(링크)의 첫글자 ref는 reference참조하다의 앞 3글자
저 주소를 참조해라 target은 새 창으로 뛰우고 싶을 때 title은 링크를 클릭하기 전 설명
